# MaumTalk_MVP

5개의 정적 HTML 프로젝트를 하나의 실행 가능한 MVP 폴더로 통합한 버전입니다.

## 구조

- `index.html`: Home으로 진입
- `router.js`: 페이지 이동 및 Bottom Navigation 연결
- `storage.js`: 공통 localStorage 유틸리티
- `assets/`: 공통 asset 보관
- `pages/home/`: Home 원본
- `pages/editor/`: Editor 원본
- `pages/today/`: Today 원본
- `pages/tuklog/`: Tuklog 원본
- `pages/my/`: My 원본

## 실행

`index.html`을 브라우저로 열면 됩니다.
